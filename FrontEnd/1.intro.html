HTML 학습하기

h태그는 제목을 의미하고 위 아래로 약간의 여백이 자동 삽입

검색엔진에서 h태그는 중요한 검색 키워드로 사용

p태그는 단락(paragraph)의미하고, 태그의 위 아래로 여백이 삽입

html 태그 내에서는 엔터를 치더라도 줄바꿈이 되지 않고,
br 태그를 써야한다. 도한 웹 브라우저에서는 여러번 띄어쓰기를 하더라도 한번만 띄어쓰기 됨에 유의

텍스트의 형식화

굵게는 b 태그

강조는 strong 태그

이텔릭체는 i 태그

강조된 이텔릭체는 em 태그

하이라이트 텍스트는 mark 태그

삭제된 텍스트는 del 태그
